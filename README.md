# learnFinance 📚 金融投资学习指南

> 从零开始系统学习金融投资知识，涵盖股票、基金、债券、期货、期权等主流投资品种。

---

## 📖 项目简介

本项目是一个**系统化的金融投资学习笔记库**，旨在帮助金融小白：
- ✅ 理解各类投资产品的底层逻辑
- ✅ 掌握实战操作技巧与风险控制
- ✅ 避开常见的投资陷阱
- ✅ 建立科学的资产配置理念

**学习理念**：理论与实践结合 + 风险意识优先 + 长期价值投资

## 📚 学习路径导航

### 🟢 初级：现金管理工具（零基础可学）

#### 💰 货币基金
**定位**：现金管理的最佳工具  
**核心优势**：安全性极高、流动性好、门槛低

📖 **学习内容**：
- 货币基金会亏损吗？
- 余额宝、零钱通有何区别？
- "七日年化"和"万份收益"是什么意思？
- 什么时候收益会变高？

👉 [开始学习：货币基金入门](./基金/货币基金/货币基金入门.md)

---

#### 🔄 国债逆回购
**定位**：账户闲钱的安全港  
**核心优势**：交易所担保、国债抵押、短期高收益

📖 **学习内容**：
- 国债逆回购的核心概念
- 操作指南与常见纠错
- 月末、季末、节假日套利技巧

👉 [开始学习：国债逆回购](./债券/国债逆回购/国债逆回购.md)

---

### 🟡 中级：稳健投资工具（适合保守型投资者）

#### 📊 债券基金
**定位**：稳健收益的"固收+"工具  
**核心优势**：收益稳定、风险较低、专业管理

📖 **学习内容**：
- 债券基金为什么有时会亏损？
- 什么时候是买入的好时机？
- 如何挑选优秀的债券基金？
- 家庭资产配置中的角色定位

👉 [开始学习：债券基金入门](./基金/债券基金/债券基金入门.md)

---

#### 📈 指数基金ETF
**定位**：打包一篮子股票的投资工具  
**核心优势**：分散风险、费用低、透明度高

📖 **学习内容**：
- ETF为什么总能买到（申购-赎回机制）
- ETF的溢价套利原理
- 跨境ETF的时区套利
- 如何利用盘前指标做"天气预报"

👉 [开始学习：指数基金ETF](./基金/指数基金/指数基金ETF.md)

---

### 🟠 高级：进取型投资工具（需要专业知识）

#### 💹 股票
**定位**：上市公司的所有权凭证  
**核心优势**：长期回报高、可参与公司成长

📖 **学习内容**：
- 股票价格形成机制
- 如何看懂分时图和K线图
- 价值投资 VS 趋势交易
- 仓位管理与止损纪律
- 常见投资陷阱与避坑指南

👉 [开始学习：股票投资基础](./股票/股票投资基础.md)

---

### 🔴 专家级：杠杆衍生品（高风险高收益）

#### ⚡ 期货
**定位**：以小博大的双刃剑  
**核心优势**：双向交易、T+0、高杠杆

⚠️ **风险警示**：保证金制度、爆仓风险、杠杆放大效应

📖 **学习内容**：
- 期货的杠杆和保证金机制
- 做空赚钱的具体操作
- 期货交割与移仓换月
- 新手生存法则与风险管理
- 主要期货品种介绍

👉 [开始学习：期货交易基础](./期货/期货交易基础.md)

---

#### 🎯 期权
**定位**：金融市场的"保险"工具  
**核心优势**：买方风险有限收益无限、策略灵活多样

⚠️ **风险警示**：时间价值衰减、波动率风险、卖方风险极高

📖 **学习内容**：
- 认购期权 VS 认沽期权
- 内在价值与时间价值
- 买方与卖方的风险收益
- 希腊字母（Delta、Gamma、Theta、Vega）
- 牛市价差、跨式、铁鹰式等组合策略

👉 [开始学习：期权交易基础](./期权/期权交易基础.md)

## 💡 投资建议与资产配置

### 🎯 根据风险承受能力选择品种

| 风险类型 | 推荐配置 | 核心策略 |
|---------|---------|---------|
| **极度保守** | 货币基金60% + 债券基金30% + 国债逆回购10% | 保本为主，跑赢通胀 |
| **保守稳健** | 债券基金50% + 指数基金30% + 货币基金20% | 稳健增值，控制回撤 |
| **平衡配置** | 指数基金40% + 债券基金30% + 股票20% + 货币基金10% | 攻守兼备，长期复利 |
| **积极进取** | 股票50% + 指数基金30% + 债券基金15% + 货币基金5% | 追求高收益，承受波动 |
| **激进投机** | 股票60% + 期货/期权20% + 指数基金15% + 货币基金5% | 高风险高收益，专业操作 |

### 📌 黄金法则

1. **永远不要满仓**：保留至少20%现金应对风险和机会
2. **分散投资**：不把鸡蛋放在同一个篮子里
3. **定期再平衡**：每季度或半年调整一次资产配置
4. **止损铁律**：设置止损线并严格执行
5. **持续学习**：市场在变化，你的认知也要与时俱进

## 🚀 推荐学习顺序

### 阶段一：建立基础（1-2周）
1. 了解各类投资品种的基本概念
2. 学习货币基金和国债逆回购的使用
3. 开始打理闲置资金

### 阶段二：稳健投资（1-3个月）
1. 深入学习债券基金的配置方法
2. 掌握指数基金的投资策略
3. 开始定投实践

### 阶段三：进阶实战（3-6个月）
1. 系统学习股票投资知识
2. 建立自己的交易系统
3. 小资金实盘练习

### 阶段四：高阶探索（仅限有基础者）
1. 模拟期货/期权交易至少3个月
2. 深入理解杠杆和风险管理
3. 小仓位谨慎尝试
